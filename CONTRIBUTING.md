# Contributing

## Setup
- Python 3.10+
- Install: `pip install -r requirements.txt && pip install -r requirements-dev.txt`
- Editable (optional): `pip install -e .`

## Commands
- Lint: `ruff check .`
- Tests: `pytest -q`
